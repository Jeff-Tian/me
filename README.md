# me

> 我的个人中心

##安装与运行

首先需要全局安装 cli 开发工具 @tarojs/cli

```shell
$ npm i -g @tarojs/cli
$ yarn global add @tarojs/cli
```

### 微信小程序开发

然后执行如下命令，运行完后打开微信开发者工具，选择项目根目录进行预览

```
$ taro build --type weapp --watch
```
